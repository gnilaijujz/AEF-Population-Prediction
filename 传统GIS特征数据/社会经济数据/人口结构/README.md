# 社会经济:人口结构

> 第一块 · 社会经济 / 人口结构　|　整理自 `人口结构_数据说明.docx`
> 输出:`pop_features_15msa.csv`

## 一、数据概况

| 项 | 内容 |
|----|------|
| 数据来源 | ACS 5-year 2016–2020(表 DP05,data.census.gov / ACSDT5Y2020) |
| 时间 | 2016–2020 五年合并估计(一个值,非逐年) |
| 尺度 | census tract |
| 连接键 | `cb_2020_3` |
| 是否归一化 | 否 |

## 二、特征字段

| 字段 | 含义 / 计算 | 范围 | 数值大代表 |
|------|-----------|------|-----------|
| `minor_ratio` | 未成年占比 `DP05_0019E / DP05_0001E` | 0~1 | 未成年比例越高 |
| `elderly_ratio` | 老年占比 `DP05_0024E / DP05_0001E` | 0~1 | 老龄化越高 |
| `sex_ratio` | 男/女 `DP05_0002E / DP05_0003E ×100` | ≈100 附近 | 男性相对越多 |

## 三、说明与注意

- `sex_ratio` 有极端离群值(监狱/军营/宿舍等集体宿舍性别失衡),建模前可截断或取对数。
- 0 人口 / 0 女性 tract 算不出 → 留空,未插补。
- 本表 11,179 行(ACS 全口径);并入 11,158 总表时取交集。
