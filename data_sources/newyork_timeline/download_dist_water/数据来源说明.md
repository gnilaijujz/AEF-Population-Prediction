# 水体距离 `dist_water_m` · 数据来源说明（GEE 路线）

> 配套脚本：`dist_water_export_gee.js`
> 用途：为 New York MSA 导出 **2017 / 2018 / 2019 / 2020 / 2021** 年 tract 级 `dist_water_m`，
> 口径对齐 2020，支撑"空间迁移性 → 时间迁移性"对照研究。

---

## 一、数据来源

| 项 | 内容 |
|----|------|
| 产品 | **JRC Global Surface Water v1.4 · 逐年水体分类（YearlyHistory）** |
| 提供方 | European Commission JRC（经 GEE） |
| GEE 资产 | `JRC/GSW1_4/YearlyHistory`（波段 `waterClass`，逐年一层） |
| 水体定义 | `waterClass ≥ 2`（季节性 + 永久水）计为水 |
| 原生分辨率 | 30m |
| 年份覆盖 | 1984–**2021**，本研究用 2017–2021（2022+ 无原生逐年水体，需冻结用 2021 并注明） |

> GEE 无现成"距水距离"栅格，用水体掩膜 + `fastDistanceTransform` 现算。

---

## 二、处理口径（与 2020 一致：30m/90m combine）

1. **几何冻结**：用 2020 纽约 tract 矢量（键 `cb_2020__3`），各年只换水体年份、不换边界。
2. **距水距离图**：该年水体二值图 `fastDistanceTransform(512).sqrt() × 像元边长` → 到最近水像元的距离（米）。
3. **两个尺度各求 tract 均值**（同一距离表达式，只靠 `reduceRegions` 的 scale 区分）：
   - `scale=30`：512×30≈15km 搜索窗，**近处精确**、远处会截断出伪值；
   - `scale=90`：512×90≈46km 搜索窗，**远处正确**、近处较粗。
4. **combine = 逐 tract `min(30m, 90m)`**：
   - 近水 tract 30m 更小更准 → 取 30m；
   - 远水 tract 30m 出几十万米伪值、90m 正确 → `min` 自动退 90m。
   - 已核实这正是 2020 `combine.csv` 的规则（"30m 有效才用、伪值退 90m"），combine 最大值约 15km、无 410km 伪值。
   - 缺值安全处理：任一尺度为空取另一个，都空则留空不插补。
5. **键与输出**：`cb_2020__3` → 输出列 `cb_2020_3`；每年一份 `water_dist_newyork_YYYY.csv`（列 `cb_2020_3, dist_water_m`）。
6. **不归一化**：输出原始值；标准化最后统一套 **2020 那把尺子**（勿逐年重标）。

> 脚本内已把 30m/90m + min 全放进 GEE，每年**直接导出一个已 combine 好的 CSV**，无需本地后处理。
> `EXPORT_RAW=true` 可额外导出 `d30_raw/d90_raw` 两列供验收自查。

## 三、五年同管线

- 脚本 `YEARS=[2017,2018,2019,2020,2021]`，**2020 也一并重算**，五年全走同一 GEE 管线、跨年逐位可比。
- 与本项目 NDVI / NTL / 不透水面的处理策略一致。

---

## 四、使用步骤

1. 打开 https://code.earthengine.google.com/ ，粘入 `dist_water_export_gee.js`。
2. 改 `TRACTS_ASSET` 为你上传的 2020 纽约 tract 资产（与 AEF/NDVI/NTL/不透水面同一个）。
3. Run → Tasks 面板出现 5 个任务 → 逐个点 RUN → 导到 Drive 的 `dist_water_export/`。
4. 首次跑先看打印的 tract 数量、第一条属性（确认键 `cb_2020__3`），并可先 `EXPORT_RAW=true` 抽查 `min`。

---

## 五、验收自查

- [ ] `dist_water_m = min(d30, d90)`；远处无几十万米伪值（combine 最大约 15km）。
- [ ] 近水 tract 接近 0；各年略有变化（水体年际变化很小，差异应很小但非全等）。
- [ ] 行/键与 2020 tract 一一匹配；缺值 tract 留空不插补；标准化用 2020 尺子。
